How to read these cards
  • Bases Empty reflects all plate appearances with no runners on base — it does not change with filters.
  • The situation column (Runners On / RISP / Loaded) responds to the situation filter above.
  • The delta pill compares runners-on vs. bases-empty performance. Positive means the batter elevates with runners on.
  • Clutch Factor sort weights the delta by sample size — small PA counts are pulled toward zero.
  • The PA count reflects the number of PAs that match the filter criteria (season, outs, innings, and situation). It does NOT include PAs where no runners were on base.
